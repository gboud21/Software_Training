# Guidelines
## Principal 1
- Write clear and specific instructions
    1. Use delimiters to clearly frame the prompts to reduce misinterpertation
        1. ie if text contains words that could change the output of the prompt
    1. Ask for structured output
        1. structured output can make it easier for the AI to format the data and make it easily injectable into code
    1. Check whether conditions are satisified. Check assumptions required to do the tasks
    1. Few-shot prompt
        1. Give successful examples as part of the prompt to guide it to the correct answer
## Prinicpal 2
- Give the model time to think. Just like with a person, if you try to rush AI to complete a complex task then it will likely make erros
    1. Break the task down into steps
    1. Insttruct the model to work out its own solution before rushing to a conclusion
